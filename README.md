## 🤖 AI Agent

An AI-powered chatbot web application built using **HTML, CSS, JavaScript, Flask, and OpenRouter API**. Users can enter questions through a simple web interface, and the chatbot generates intelligent responses using a Large Language Model (LLM).

---

## 🚀 Features

- 💬 Ask questions through a clean and responsive web interface
- ⚡ Real-time AI responses
- 🔗 Frontend and backend integration using Flask
- 🤖 Supports OpenRouter AI models
- 📱 Responsive and user-friendly design

---

## 🛠️ Technologies Used

- Python
- Flask
- HTML5
- CSS3
- JavaScript

## 8. Open in your browser


http://127.0.0.1:5000


---

## 💡 How It Works

1. User enters a question.
2. JavaScript sends the question to the Flask backend.
3. Flask sends the request to the OpenRouter API.
4. The selected AI model processes the question.
5. The AI response is returned and displayed on the webpage.
